# Sitecreator

Agency website, admin CRM, client portal, realtime chat, blog, media library, calculator, and service landing pages built with Next.js.

## Local Development

```bash
npm ci
npm run dev
```

Open `http://localhost:3000`.

## Quality Commands

```bash
npm run lint
npm run test
npm run build
```

## Production

This project is deployed on a VPS behind `nginx` and managed with `pm2`.

See `DEPLOY.md` for:

- production env setup
- `pm2` config
- `nginx` reverse proxy setup
- GitHub Actions CI/CD pipeline
